# Joystick Theater Mode

Joystick Theater Mode is a uBlock Origin filter list that maximizes the stream viewing area by hiding unused UI elements. It hides navbars and the stream title, then reveals the interface when you hover the stream area.

## What it does

   - Hides navbars and the stream title to reduce visual clutter.
   - Reveals UI on hover so the stream player HUD/description remain accessible.
   - Works by applying page element hiding rules that support “theater mode” behavior.

## How to install

   - Open uBlock Origin settings.
   - Go to the Filter lists tab.
   - Add/subscribe to the [filterlist.txt](https://github.com/Schokolabbi/JoystickTheaterMode/raw/refs/heads/main/filterlist.txt)
   - Refresh the stream page (or reload the tab) to apply changes.

## Usage notes

   - Theater mode is enabled once the stream player interface disappears.
   - Hovering over the stream area should bring the UI back.
